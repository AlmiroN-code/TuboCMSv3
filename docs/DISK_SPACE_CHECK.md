# Проверка свободного места на диске

## Описание

Система проверяет доступное место на диске перед началом обработки видео.

## Требования

- Минимум 1GB свободного места
- Проверка выполняется перед каждой обработкой
- Ошибка если места недостаточно

## Как работает

```python
def _check_disk_space(self, required_gb=1):
    total, used, free = shutil.disk_usage(settings.MEDIA_ROOT)
    free_gb = free / (1024**3)
    
    if free_gb < required_gb:
        raise Exception(f"Insufficient disk space: {free_gb:.1f}GB available, {required_gb}GB required")
```

## Преимущества

- Предотвращение ошибок "disk full"
- Раннее обнаружение проблем с местом
- Защита системы от переполнения диска
- Информативные сообщения об ошибках

## Настройка

Можно изменить минимальное требование в коде сервиса.
