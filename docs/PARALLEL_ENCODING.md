# Параллельное кодирование видео

## Описание

Система кодирует несколько профилей видео одновременно для ускорения обработки.

## Как работает

- Использует ThreadPoolExecutor с max_workers=2
- Каждый профиль кодируется в отдельном потоке
- Результаты собираются через futures

## Производительность

- Ускорение до 2x при обработке нескольких профилей
- Оптимальное использование CPU ресурсов
- Автоматическое управление нагрузкой

## Настройка

В `apps/videos/services/video_encoder.py`:

```python

    with ThreadPoolExecutor(max_workers=max_workers) as executor:
        # Параллельная обработка
```

## Ограничения

- Максимум 2 потока (чтобы не перегружать систему)
- Требует достаточно RAM для нескольких процессов FFmpeg
